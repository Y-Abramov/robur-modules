# robur-modules

Каталог плагинов для [Topomatic Robur](https://topomatic.ru) от [Yaroslav I. Abramov](https://abrmove.ru).

## Установка

1. Скачать и установить **[AbrModules](https://github.com/Y-Abramov/abrmodules/releases/latest)** — менеджер плагинов
2. В Robur открыть **Библиотека модулей** и установить нужные плагины

## Плагины

| Плагин | Описание | Совместимость |
|--------|----------|---------------|
| **Библиотека модулей** | Установка и обновление плагинов ABR | Все продукты Robur |
| **Диспетчер стилей** | Управление конфигурациями слоёв и стилями дорог | Автомобильные дороги |
| **Отслеживание курсора** | Синхронизация курсора между планом и профилем | Автомобильные дороги, Железные дороги |
| **Диспетчер моделей** | Управление моделями проекта, массовое переименование | Все продукты Robur |
| **Быстрые команды** | Палитра быстрого запуска команд Robur с поиском | Все продукты Robur |

## Требования

- Topomatic Robur сборка **16.0.62.12** и выше
